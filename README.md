# Känslokartan

Emotion recognition and journal for autism and ADHD.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install kanslokartan
```

### Fedora/RPM
```bash
sudo dnf install kanslokartan
```
